# Pandas CSV Analysis  
Syntecxhub Data Science Internship – Week 1  

## 📌 Objective
Perform basic data analysis and cleaning using Pandas on the Titanic dataset.

## 🔧 Tasks Performed
- Loaded CSV dataset using Pandas
- Performed data inspection (info, describe, missing values)
- Handled missing values (median & mode imputation)
- Dropped irrelevant column (Cabin)
- Filtered passengers above 30 years
- Selected important columns
- Exported filtered dataset to CSV

## 📊 Key Insights
- Total passengers: 891
- Average age ≈ 29 years
- Maximum fare ≈ 512
- 305 passengers are older than 30
- 314 passengers are female

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
